# Adventure Park Sales Analysis Project

## Project Overview
The Adventure Park Sales Analysis Project is a business data analysis project focused on understanding sales performance, profitability, regional trends, reseller performance, and employee efficiency using Excel.

The project involves complete data preprocessing, exploratory data analysis (EDA), KPI analysis, and dashboard development using a real-world B2B transactional dataset containing more than 57,000 sales records.

---

# Objectives
The main objectives of this project are:

- Analyze overall business sales performance
- Identify high-performing and low-performing products
- Track regional and country-wise sales trends
- Evaluate employee sales performance
- Compare sales targets vs achievements
- Analyze reseller contribution to revenue
- Calculate profitability and cost variance
- Generate business insights for decision-making

# Key Calculations

## Profit
Profit = Sales – Cost

## Profit Margin %
Profit Margin % = (Profit ÷ Sales) × 100

## Cost Variance
Cost Variance = Cost – Standard Cost

## Target Achievement %
Target Achievement % = (Actual Sales ÷ Target) × 100

# Business Insights

Some major insights identified from the project:

- Profit margins vary across product categories
- Some resellers consistently generate high revenue
